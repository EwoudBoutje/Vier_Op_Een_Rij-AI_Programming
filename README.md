# Welkom bij Vier Op Een Rij
## Hoe spelen?
-> download Connect4_V2.exe en voer het uit!

## Files in de repo
- main.py: vier op een rij classic (player vs player, zonder menu)
- vier_op_een_rij_minimax.py: vier op een rij classic met AI (player vs AI, zonder menu, nog steeds dom)
- Connect4_game.py: vier op een rij: The Game (Player vs AI, Mooi Menu, een echte game, verschillende formaten, maar nog problemen met AI bij de grotere formaten)
- Connect4_game_v2.py: vier op een rij: The Game V2 (Player vs AI, Mooi Menu, een echte game, verschillende formaten, AI is ook slim bij andere formaten maar soms ook zeer dom)

- Connect4_basic.py: vier op een rij classic met AI (vier_op_een_rij_minimax maar een beetje anders)

- Connect4_basic_V2.py: vier op een rij classic met AI(player vs AI, zonder menu,bijna onverslaanbare Ai)
